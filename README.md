# Tadss

- O que é MUTEX?
- Diferença entre paralelismo e concorrencia  (concorrencia é usada no dia-a-dia)
- O que é Sleep?
- Qualquer objeto no java pode ser usado pra fazer synchronized
- Lock acontece quando duas threads executam ao mesmo tempo
- Diferença entre mutex e semaforo
    A principal diferença entre Mutex e Semáforo é que o Mutex é um mecanismo de bloqueio para exclusão mútua, permitindo apenas uma thread por vez em um recurso, enquanto o Semáforo é um mecanismo de sinalização que permite acesso controlado a múltiplas instâncias de um recurso (contagem).
